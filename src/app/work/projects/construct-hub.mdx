## Overview

Construct Hub is a comprehensive web platform developed as a second-year university project. The application connects clients and customers with a skilled workforce, heavy-duty vehicles, and rental equipment services. It acts as a digital bridge to address industry challenges and help companies achieve their objectives efficiently.

## Key Features

- **Service Marketplace**: Allows users to discover skilled workers, vehicles, and equipment through a centralized platform.
- **Simple and Intuitive UI**: Designed to ensure easy navigation and quick access to services.
- **MVC-Based Architecture**: Structured using the Model–View–Controller pattern for better separation of concerns.
- **Dynamic Content Handling**: Enabled server-side processing for service listings and user interactions.

## Technologies Used

- **HTML, CSS, JavaScript**: For building the frontend interface.
- **PHP**: For backend logic and server-side processing.
- **MVC Architecture**: To improve maintainability and scalability of the application.

## Challenges and Learnings

A key challenge was designing a structured backend using MVC while maintaining a simple and user-friendly frontend. This project helped me gain a solid understanding of web fundamentals, backend development with PHP, and architectural design patterns.

## Outcome

Construct Hub provided a practical solution for connecting construction-related services through a single platform. The project strengthened my foundational skills in web development and system architecture and laid the groundwork for more advanced full-stack projects.
