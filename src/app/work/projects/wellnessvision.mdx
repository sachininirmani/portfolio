## Overview

WellnessVision is a smart web platform developed as a third-year university project, aimed at helping individuals achieve a balanced lifestyle by promoting mental and physical well-being. The platform connects health professionals, volunteers, and the general public through a single digital interface, making wellness support more accessible and reducing everyday stress.

## Key Features

- **User-Centered Wellness Platform**: Enables users to connect with health professionals and volunteers for guidance and support.
- **Role-Based Access**: Designed separate user flows for health professionals, volunteers, and the public.
- **Responsive Web Interface**: Built with a focus on usability and accessibility across devices.
- **Client–Server Communication**: Implemented a clear separation between frontend and backend using REST-based interactions.

## Technologies Used

- **React & TypeScript**: For building a scalable and interactive frontend.
- **CSS & Bootstrap**: For responsive styling and layout consistency.
- **Java & Spring Boot**: For backend development and API implementation.
- **Client–Server Architecture**: Ensuring modularity and maintainability.

## Challenges and Learnings

One of the main challenges was designing a system that balances usability with functionality for different user roles. This project strengthened my understanding of full-stack development, RESTful APIs, and client–server architecture, as well as teamwork and requirement analysis.

## Outcome

WellnessVision successfully demonstrated how technology can support mental and physical well-being through accessible digital solutions. The project enhanced my skills in full-stack development and system design while addressing a meaningful real-world problem.
